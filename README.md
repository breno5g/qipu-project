# MEI GRATIS

## Como rodar o projeto?

### Requisitos:

- node >= 16.15.0

### Comandos

```bash
  npm install
  npm run dev
```

Em seguida acesse http://localhost:3000

## Tecnlogias utilizadas:

- Typescript
- React
- Vite
- React router dom
- Styled components
- Eslint
- Prettier
- React icons
- Sass
- React select
